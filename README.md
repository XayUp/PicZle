# PicZle
Divida uma imagem em várias parte e exporte de uma só vez!

## Funcionamento
Este aplicativo permite que você corte uma imagem em várias partes (Quebra cebeça) e exporte em imagens divididas, ou em um PDF único.

## Objetivo
Este é mais um dos aplicativo que já existe que fazem a mesma coisa, mas decidi fazer algum muito mais simples e com muito mais controle, pois tenho convivência com pessoas que tem necessidade de algo simples, prático e rápido, além de que posso ter sugestões diretas do que pode ou não ter útil e aperfeiçoar ainda mais o aplicativo.

## Construindo
Após ter o SDK Flutter instalado na sua máquina, clone este repositório:
```bash
git clone https://github.com/XayUp/PicZle.git
cd PicZle
```
compile para a plataforma de destino, como o Android por exemplo:
```bash
flutter build apk --release
```
